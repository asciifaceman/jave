# Loops Guide

Jave intentionally provides dual `given` loop syntax in v0.1.

## While-ish

```jave
given (<X lesslysame 3>) again -> {
    X 2b=2 X + 1;;
}
```

## For-ish

```jave
given (<allow exact I 2b=2 0;; I lessly 3;; I 2b=2 I + 1;;>) -> {
    pront(I);;
}
```

## Collection Iteration

```jave
given (<Name within Names>) -> {
    pront(Name);;
}
```

See `examples/loops/main.jave`.

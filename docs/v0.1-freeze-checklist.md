## Known Limitations and Non-Goals for v0.1

### Intentional Omissions
- No generics or parametric types
- No user-defined types (structs, classes, enums)
- No closures or first-class sequences
- No exception handling (only `Panic` builtin)
- No standard library beyond builtins
- No debugger or REPL
- No module versioning system
- No package manager integration
- No LSP or editor tooling beyond syntax highlighting

### Known Bugs and Edge Cases
- [ ] **Windows terminal color support**: Not validated on older Windows versions (<10)
- [ ] **Large file handling**: sourceIndex encoding assumes <1M lines per file
- [ ] **Error recovery limits**: Parser may produce incomplete ASTs after certain syntax errors
- [ ] **Import path ambiguity**: Workspace root heuristic may fail in monorepo scenarios

### Deferred to Post-v0.1
- Optimization passes (dead code elimination, constant folding)
- Incremental compilation
- Watch mode for baggage
- Binary distribution / release packaging
- Comprehensive benchmark suite
